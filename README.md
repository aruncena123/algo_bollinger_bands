# A short program to buy and sell assets based on bollinger bands

1. The above program is to automate the trading of cryptocurrency, based on the signal indicator of bollinger bands(**BB**).
2. The following code has 3 main takeaways:
    * code written in: **Python**
    * medium for back testing : **Enigma Catalyst**
    * librarys used: 
        * **TA-lib**
        * matplotlib
        * pandas
3. To repruduce the above results use:
    * [anaconda](https://www.anaconda.com/download/) or [miniconda](https://conda.io/en/master/miniconda.html) (to create a sandbox environment)
    * [enigma catalyst](https://github.com/enigmampc/catalyst)
    * [ta-lib](https://pypi.org/project/TA-Lib/) or `pip install TA-Lib`
